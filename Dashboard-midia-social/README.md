## 🎯 Objetivo do Projeto

Este projeto tem como objetivo analisar o impacto das redes sociais no bem-estar de estudantes, utilizando técnicas de análise de dados e visualização por meio do Power BI.

A proposta consiste em transformar dados brutos em informações relevantes, permitindo identificar padrões e relações entre o uso de redes sociais, saúde mental, qualidade do sono e outros fatores.

---

## 📁 Base de Dados

O conjunto de dados utilizado foi o arquivo:

```
Social_media_impact_on_life.csv
```

Este dataset contém informações sobre:

* Tempo médio de uso diário de redes sociais
* Pontuação de saúde mental
* Horas de sono por noite
* Plataforma mais utilizada
* Nível acadêmico
* Impacto geral (positivo, neutro ou negativo)

---

## 🛠️ Ferramentas Utilizadas

* Power BI Desktop
* Microsoft Excel / CSV
* Conceitos de Análise de Dados (Pandas - abordagem conceitual)

---

## 📊 Estrutura do Dashboard

### 🔹 Página 1 - Visão Geral

Nesta página são apresentados gráficos que permitem uma análise geral dos dados:

* Distribuição do impacto geral nos estudantes
* Relação entre tempo de uso diário e saúde mental
* Comparação das horas de sono com o impacto geral
* Análise das plataformas mais utilizadas

Também foram implementados filtros interativos, como:

* Impacto geral
* Nível acadêmico
* Gênero

---

### 🔹 Página 2 - Análise de Influenciadores

Nesta seção foi utilizado o recurso de **Principais Influenciadores** do Power BI para identificar os fatores que mais impactam:

* Impacto negativo na vida dos estudantes
* Redução da saúde mental
* Diminuição das horas de sono

Essa análise permite compreender quais variáveis possuem maior influência nos resultados observados.

---

## 📈 Principais Insights

* O aumento no tempo de uso diário das redes sociais está associado à piora na saúde mental
* Menores horas de sono aumentam a probabilidade de impactos negativos
* O uso excessivo de redes sociais pode afetar diretamente o bem-estar dos estudantes
